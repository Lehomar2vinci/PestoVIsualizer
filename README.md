## 1. English (Anglais)

# Dynamic Audio Visualizer

This project is an interactive audio visualizer using the [p5.js](https://p5js.org/) library to create dynamic visualizations based on microphone input.

## Features

- **Audio Visualization**: Concentric circles that change size and color based on detected audio frequencies.
- **Animated Particles**: Enable or disable animated particles driven by frequency energy.
- **Color Palette**: Choose from different color palettes (default, warm, cool).
- **Adjustable Sensitivity**: Fine-tune the audio visualization sensitivity.

## Installation

1. **Clone the repository**:

    ```bash
    git clone https://github.com/Lehomar2vinci/PestoVIsualizer
    ```

2. **Navigate to the project directory**:

    ```bash
    cd PestoVIsualizer
    ```

## Usage

1. Open **listener.html** in your browser.
2. Use the on-screen controls to adjust sensitivity, select a color palette, and enable particles.
3. Click **"Start"** to activate microphone input and begin visualization.

### Having Issues?
If you encounter errors (e.g., the microphone not working), make sure:
- Your browser has access to the microphone (check browser permissions).
- You are running a secure context (HTTPS) if required by your browser.

For more help, consult the [p5.js documentation](https://p5js.org/get-started/) or the [p5.sound.js docs](https://p5js.org/reference/#/libraries/p5.sound).

## Main Files

- **listener.html**: Contains the HTML structure and UI controls.
- **listener.css**: Contains the CSS styling for the user interface.
- **listener.js**: Contains the JavaScript logic for audio visualization and particle animation.

## Dependencies

- **[p5.js](https://p5js.org/)**: A JavaScript library for creative coding and interactivity.
- **[p5.sound.js](https://p5js.org/reference/#/libraries/p5.sound)**: An extension of p5.js for handling sound input and analysis.

このプロジェクトは、[p5.js](https://p5js.org/) ライブラリを使用して、マイク入力に基づく動的なビジュアライゼーションを作成するインタラクティブなオーディオビジュアライザーです。

- **カラーパレット**: デフォルト、ウォーム、クールなど、さまざまなカラーパレットを選択可能。
- **感度調整**: オーディオビジュアライゼーションの感度を調整可能。

1. **リポジトリをクローン**:

    ```bash
    git clone https://github.com/Lehomar2vinci/PestoVIsualizer
    ```

2. **プロジェクトディレクトリに移動**:

    ```bash
    cd PestoVIsualizer
    ```

- **颜色调色板**: 可选择不同的颜色调色板（默认、暖色、冷色）。
- **可调敏感度**: 调整音频可视化的敏感度。
